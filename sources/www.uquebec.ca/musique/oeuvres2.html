Catalogues found on this site

Cette section contient, pour un certain nombre de compositeurs de musique classique, un catalogue complet de leurs oeuvres basé, lorsque disponible, sur un ou des catalogues thématiques.

Il faut noter que pour chacun des compositeurs dont le nom appararaît dans la liste ci-dessous et, pour lesquels aucun renvoi ultérieur n'est inclus dans ces pages, le catalogue des oeuvres est complété dans ma banque de données ACCESS sur ordinateur et chacun sera progressivement ajouté tout en tenant compte de l'espace-disque requis.
This section contains, for a certain number of classical music composers, a complete catalogue of their works based, if available, on thematic catalogues. You might notice that all the information is in French, it is because the research work was done in French. I do not intend to translate it completely in English but translation notes will be added where applicable.

For composers whose name appears in the following list and for which there is no further cross-reference in these pages, the catalogue is only available on my working station in an ACCESS database and the content will be added on to these pages depending on space requirement/availability on the server.
